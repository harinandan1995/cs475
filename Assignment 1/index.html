How modelling mode works

The program is by default set in modelling mode. You can start creating your model by clicking "left mouse button" on the screen to add vertices. You see nothing till you add the 3rd vertex. You can remove the last added point by pressing "shift + left mouse button". You can save the file by pressing "K key" and entering the name in the teminal. You can also load a raw file by pressing "L key" and entering the file name in the terminal. When you load a object the mode is set to modelling by default.

    Key controls

  • Left mouse click to add a point to your model
  • Shift + left mouse click to remove last added point in your model
  • K key to save the model into a raw file
  • L key to load a raw file
  • I key to change mode to inspection mode
  • N key to create a new model

How inspection mode works

You can change the mode from modelling to inspection by pressing "I key". In inspection mode you can rotate the model about its centroid, you can tranlate the object along +ve and -ve x,y,z axes. The controls are mentioned below. By pressing "R key" the origin is moved to the centriod of the object.

    Key controls

  • Left,Right arrows to rotate about Y-axis
  • Up,Down arrows to rotate about X-axis
  • PgUp,PgDn to rotate about Z-axis
  • R key to move origin to centroid
  • W,A keys to tranlate along Y-axiz
  • A,D keys to tranlate along Z-axiz
  • Z,X keys to tranlate along Z-axiz
  • M key to change mode to modelling mode
  • N key to create a new model

Assumptions

The following are the assumptions made as part of this assignment

  • Program start with modelling mode
  • The model is shown when more than 2 vertices are clicked
  • Saving a raw file overwrites already existing raw file with same name
  • Raw file will be empty if you try to save a model with less than 3 vertices
  • File used to load is a valid raw file
  • Everytime we load a file the vertex/color data is stored in one vao and the new data is stored in a seperate vao
  • When we load a file the unsaved data of previous file is lost (no prompt is shown)
